Pair this browser

Open the Nightshade desktop application and watch its console output. A new 6-digit code will be printed there. Type it in below and press Pair to complete first-run setup. The code expires after a few minutes; if it does, click Pair again to request a fresh one.

Polar alignment

1 2 3

Choose the alignment method. TPPA rotates the mount in RA between two plate-solves to compute the polar axis offset. All-sky (Sharpcap style) takes one solve from the current pointing and computes the offset directly — useful when the polar region is obstructed.

Imaging settings used between solves.

Press Start to begin. Progress updates appear in the Mount panel below this dialog; closing the modal does not stop the alignment.
